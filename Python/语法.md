

## 命名规范

#### 变量和函数

用全小写，单词之间用下划线分隔，例如：`my_variable`、`calculate_sum`。

### 类名

**采用驼峰命名法（CapWords或PascalCase）**：每个单词首字母大写，不使用下划线，例如：`MyClass`、`DataProcessor`。

### 常量

**全部大写，单词间用下划线**：例如：`MAX_SIZE`、`DEFAULT_TIMEOUT`。

### 模块和包

**全部小写**：通常用短小的名字，无下划线，例如：`utils.py`、`datahandler.py`。

## Python中for循环的使用介绍

#### 基本语法

```python
for 变量 in 可迭代对象:
    # 循环体代码块
```

#### 基本用法示例

##### 1. 遍历列表

```python
fruits = ['apple', 'banana', 'cherry']
for fruit in fruits:
    print(fruit)
```

##### 2. 遍历字符串

```python
for char in "Hello":
    print(char)
```

#### 3. 使用range()函数

```python
# 打印0到4
for i in range(5):
    print(i)

# 打印5到9
for i in range(5, 10):
    print(i)

# 打印0到10，步长为2
for i in range(0, 10, 2):
    print(i)
```

#### 4. 使用获取索引和值

```python
fruits = ['apple', 'banana', 'cherry']
for index, fruit in enumerate(fruits):
    print(f"索引 {index}: {fruit}")
```

#### 5.遍历字典

```python
person = {'name': 'Alice', 'age': 25, 'city': 'New York'}

# 遍历键
for key in person:
    print(key)

# 遍历键值对
for key, value in person.items():
    print(f"{key}: {value}")
```

#### 6.使用zip()并行遍历多个序列

```python
names = ['Alice', 'Bob', 'Charlie']
ages = [25, 30, 35]
sex = ['F', 'M', 'F']

for name, age, s in zip(names, ages, sex):
    print(f"{name} is {age} years old and is a {s}")
```

#### 7.列表推导式中的for循环

```python
# 生成平方数列表
squares = [x**2 for x in range(10)]
print(squares)
```

## 存放数据的数据结构介绍

#### 1. 列表 (List)

```python
# 创建列表
fruits = ['apple', 'banana', 'cherry']
numbers = list(range(5))  # [0, 1, 2, 3, 4]

# 访问元素
print(fruits[0])  # 'apple' (索引从0开始)
print(fruits[-1])  # 'cherry' (负索引表示从末尾开始)

# 切片操作
print(numbers[1:3])  # [1, 2] (包含1，不包含3)
print(numbers[::2])  # [0, 2, 4] (步长为2)
```

**常用方法**

```python
# 添加元素
fruits.append('orange')  # 末尾添加
fruits.insert(1, 'grape')  # 在索引1处插入

# 删除元素
fruits.remove('banana')  # 删除第一个匹配项
popped = fruits.pop(1)  # 删除并返回索引1的元素

# 其他操作
fruits.extend(['pear', 'kiwi'])  # 合并列表
fruits.sort()  # 原地排序
sorted_fruits = sorted(fruits, reverse=True)  # 返回新排序列表
print(len(fruits))  # 获取长度
```

#### 2. 元组 (Tuple)

```python
# 创建元组
coordinates = (10.0, 20.5)
rgb = tuple([255, 128, 0])  # (255, 128, 0)

# 访问元素
print(coordinates[0])  # 10.0
x, y = coordinates  # 解包赋值

# 不可变性
# coordinates[0] = 15.0  # 会报错，元组不可变

# 特殊单元素元组

```

#### 3. 集合 (Set)

```python
# 创建集合
unique_nums = {1, 2, 3, 3, 4}  # {1, 2, 3, 4}
vowels = set(['a', 'e', 'i', 'o', 'u', 'u'])  # 去重
print(vowels)
# 添加删除元素
vowels.add('y')  # 添加元素
vowels.discard('x')  # 安全删除(不存在不报错)
vowels.remove('a')  # 删除元素(不存在会报错)

# 集合运算
a = {1, 2, 3}
b = {3, 4, 5}
print(a | b)  # 并集 {1, 2, 3, 4, 5}
print(a & b)  # 交集 {3}
print(a - b)  # 差集 {1, 2}
```

#### 4. 字典 (Dictionary)

```python
# 创建字典
person = {'name': 'Alice', 'age': 25}
squares = {x: x*x for x in range(5)}  # {0:0, 1:1, 2:4, 3:9, 4:16}

# 访问元素
print(person['name'])  # 'Alice'
print(person.get('height', 170))  # 170 (默认值)

# 添加修改
person['height'] = 165  # 添加/修改
person.update({'age': 26, 'city': 'New York'})  # 批量更新

# 删除元素
del person['age']
popped = person.pop('city')  # 删除并返回值

# 遍历
for key in person:  # 遍历键
    print(key, person[key])

for key, value in person.items():  # 同时遍历键值
    print(f"{key}: {value}")
```

#### 5. 字符串 (String)

#### 6. 高级数据结构

##### collections模块

```python
from collections import deque, Counter, defaultdict, OrderedDict, namedtuple

# 双端队列
dq = deque([1, 2, 3])
dq.appendleft(0)  # 左侧添加
dq.pop()  # 右侧删除

# 计数器
cnt = Counter(['a', 'b', 'a', 'c'])
print(cnt['a'])  # 2

# 默认字典
dd = defaultdict(int)
dd['count'] += 1  # 自动初始化为0

# 具名元组
Point = namedtuple('Point', ['x', 'y'])
p = Point(10, 20)
print(p.x, p.y)  # 10 20
```

#### 选择指南

1. **需要有序、可变序列** → 列表
   - 适合存储相同类型的数据集合
   - 示例：学生名单、传感器读数序列
2. **需要有序、不可变序列** → 元组
   - 适合固定数据集合，如坐标、数据库记录
   - 示例：RGB颜色值、函数返回多个值
3. **需要唯一元素、快速成员检测** → 集合
   - 适合去重、集合运算
   - 示例：网站唯一访问者IP、关键词集合
4. **需要键值对映射** → 字典
   - 适合快速查找、关联数据
   - 示例：配置文件、数据库记录
5. **需要高效两端操作** → deque
   - 适合队列、栈实现
   - 示例：任务队列、浏览器历史记录
6. **需要计数统计** → Counter
   - 适合频率统计
   - 示例：单词频率统计、投票计数

## 模块导入

### 1. 直接导入（import 语句）

```python
import statistics
print(statistics.median([19, -5, 36]))
print(statistics.mean([19, -5, 36]))
```

- **特点**：导入模块后，使用模块名作为前缀调用其内容（例如 `statistics.median`）。

### 2. 从模块中导入特定函数（from ... import ...）

```python
from statistics import median, mean
print(median([19, -5, 36]))
print(mean([19, -5, 36]))
```

- **特点**：只导入特定的函数或变量，可以直接调用，无需模块名前缀。

### 3. 从模块中导入所有内容（from ... import *）

```python
from statistics import *
print(median([19, -5, 36]))
print(mean([19, -5, 36]))
```

- **特点**：导入模块内所有公共成员，调用时无需任何前缀。

## 异常捕获

```python
try: # 可能产生错误的代码
    user_weight = float(input("请输 入您的体重（单位：kg）："))
    user_height = float(input("请输 入您的身高（单位：m）："))
    user_BMI = user_weight / user_height ** 2
except ValueError: 
    print("输入不为合理数字，请重新运行程序，并输入正确的数字。") # 产生值错误时会运行
except ZeroDivisionError: 
    print("身高不能为零，请重新运行程序，并输入正确的数字。") # 产生除零错误时会运行
except:
    print("发生了未知错误，请重新运行程序。") # 产生其它错误时会运行
else:
    print("您的BMI值为： " + str(user_BMI)) # 没有错误时会运行
finally:
    print("程序结束运行。") # 不管发生错误与否都会运行
```

#### 继承

子类继承父类，如果子类里没有再次写父类的方法，那么调用父类的方法，写了父类的方法就调用写了的方法

```python
class Mammal:	
    __init__(self, name, sex):
        self.name = name
        self.sex = sex
        self.num_eyes = 2

    def breathe(self):
        print(self.name + "在呼吸…")

    def poop(self):
        print(self.name + "在拉屎…")

class Human(Mammal):
    def __init__(self, name, sex, daily_salary, work_days): # 子类构造方法增加参数
        super().__init__(name, sex) # 调用父类的构造方法
        self.daily_salary = daily_salary
        self.work_days = work_days
    def read(self):
        print(self.name + "在阅读…")

class Cat(Mammal):
    def scratch_sofa(self):
        print(self.name + "在抓沙发…")
    def poop(self):
        print(self.name + "在猫砂上拉屎…")
```

## 关于if __name__ == '__main__'

#### **为什么用 `if __name__ == '__main__':`？**

- 可以确保代码**只在直接运行该文件时执行**，而**不会在作为模块导入时执行**。
- 是 Python 的**最佳实践**，避免模块被导入时意外运行测试代码。

```python
# 定义所有类和函数...

def main():
    # 运行代码写在这里
    pass

if __name__ == '__main__':
    main()
```
