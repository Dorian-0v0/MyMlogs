#### **具体例子**

```python
import matplotlib.pyplot as plt  # 导入matplotlib的pyplot模块，常缩写为plt
# 设置中文字体和 LaTeX 支持
plt.rcParams['font.sans-serif'] = ['SimHei']  # 中文字体
plt.rcParams['axes.unicode_minus'] = False    # 解决负号显示问题
plt.rcParams["text.usetex"] = False           # 禁用 LaTeX（可选，见说明）
# 准备数据
x = [1, 2, 3, 4, 5]             # 横坐标数据
y = [2, 3, 5, 7, 11]            # 纵坐标数据

# 创建一个图形窗口，设置尺寸为8x6英寸（可选）
plt.figure(figsize=(8, 6))

# 绘制折线图
# - x 和 y 为数据点
# - label 设置图例标签
# - color 指定线条颜色
# - marker 设置数据点标记样式
plt.plot(x, y, label='数据线', color='blue', marker='o')

# 添加图表标题
plt.title("示例折线图")

# 设置横坐标标签
plt.xlabel("横坐标")

# 设置纵坐标标签
plt.ylabel("纵坐标")

# 添加图例，显示标签“数据线”
plt.legend()

# 显示最终绘制的图像
plt.show()
```

#### 一个坐标系下生成多个图像

```python
import matplotlib.pyplot as plt

# 设置全局字体为支持中文的字体
plt.rcParams['font.sans-serif'] = ['SimHei']  # Windows 系统常用
# 解决负号显示问题（可选）
plt.rcParams['axes.unicode_minus'] = False

# 准备数据
x = [1, 2, 3, 4, 5]             # 横坐标数据
y1 = [2, 3, 5, 7, 11]           # 第一条曲线的纵坐标数据
y2 = [1, 4, 9, 16, 25]          # 第二条曲线的纵坐标数据（平方函数）
y3 = [1, 2, 3, 4, 5]            # 第三条曲线的纵坐标数据（线性函数）

# 创建一个图形窗口，设置尺寸为8x6英寸（可选）
plt.figure(figsize=(8, 6))

# 绘制第一条曲线（蓝色，圆形标记）
plt.plot(x, y1, label='素数序列', color='blue', marker='o', linestyle='-')

# 绘制第二条曲线（红色，方形标记）
plt.plot(x, y2, label='平方函数', color='red', marker='s', linestyle='--')

# 绘制第三条曲线（绿色，三角形标记）
plt.plot(x, y3, label='线性函数', color='green', marker='^', linestyle=':')

# 添加图表标题和坐标轴标签
plt.title("多曲线示例图")
plt.xlabel("横坐标")
plt.ylabel("纵坐标")

# 显示图例（自动根据label生成）
plt.legend()

# 显示图像
plt.show()
```
