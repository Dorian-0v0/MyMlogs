## ArcPy介绍

ArcPy 是一个 Python 站点包，它提供了一种有用且高效的方法来使用 Python 执行地理数据分析、数据转换、数据管理和地图自动化。

可以在非及时环境下使用（不使用Arcgis查看处理完的数据结果），也可以直接当成脚本使用（比如通过java调用ArcPy脚本来批量处理空间数据）
